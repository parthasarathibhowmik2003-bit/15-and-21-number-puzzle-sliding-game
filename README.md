<h1> link: https://15-and-21-number-puzzle-sliding-game.oneapp.dev/</h1><br># 15-and-21-number-puzzle-sliding-game
Table
Feature	Description
15-Puzzle	Classic 4×4 grid (numbers 1-15)
21-Puzzle	Extended 5×5 grid (numbers 1-21)
Controls	Click tiles or use Arrow Keys
Stats	Real-time move counter and timer
Visual Feedback	Green highlight when tiles are in correct position
Win Celebration	Confetti animation and victory modal
Responsive	Works on desktop and mobile devices
🎯 How to Play:
Select Puzzle Size - Click "15 Puzzle" or "21 Puzzle" to switch modes
Shuffle - Click "🔀 Shuffle" to scramble the board
Move Tiles - Click any tile adjacent to the empty space to slide it
Win - Arrange all numbers in order (left-to-right, top-to-bottom) with the empty space in the bottom-right corner
